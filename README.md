## Featured Case Studies

### Kit Pillar: Legacy Component Library Modernization
Transformed a legacy component library serving 10+ products into a modern, token-based design system. Unified fragmented components under a single design language while maintaining backward compatibility across multiple product teams.

**Technologies:** React, SCSS, Figma

### Digital Retail: Multi-Brand Design System
Architected a scalable multi-brand design system serving 5+ global airline clients. Enabled rapid brand customization without code changes, reducing theming effort from 3 weeks to 3 days per airline brand.

**Impact:** Each brand achieved unique visual identity while maintaining PROS product consistency.

## Local Development

This is a static site. To view locally:

1. Clone the repository:
   ```bash
   git clone git@github.com:rolandoff/rolandoff.com.git
   cd rolandoff.com
   ```

2. Open `index.html` in your browser, or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve
   ```

3. Navigate to `http://localhost:8000`
